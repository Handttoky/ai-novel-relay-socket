파이썬 가상환경 만들어서 프로젝트 하는 이유
ㄴ 프로젝트를 여러가지 하는 경우 각각의 프로젝트 마다 호환되는 패키지가 달라서
ㄴ 컴퓨터에 패키지 마구잡이로 깔면 오류 엄청 남 
ㄴ 그래서 가상환경에서 각각의 가상환경 마다 프로젝트에 맞는 패키지를 깔아서 관리하는 거임
ㄴ 밑의 명령어들은 전부 cmd에서 사용하는거

PyQT6를 사용하는 프로젝트를 할거임
ㄴ 비주얼 스튜디오 코드로 pythonGUI 폴더 만들고 가상환경 생성
ㄴ 터미널로 명령어 적어줌
ㄴ python -m venv 가상환경이름
ㄴ 이 프로젝트 같은 경우는 python -m virtualenv venv 라고 가상환경 만듬(virtualenv 사용)

가상환경 활성화(윈도우의 경우)
ㄴ .\venv\Scripts\activate
ㄴ 나 같은 경우에는 가상환경을 만들기 위하여 보안 오류가 떴는데     
+ CategoryInfo          : 보안 오류: (:) [], PSSecurityException
+ FullyQualifiedErrorId : UnauthorizedAccess
ㄴ Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass 요렇게 폴리시를 완화 시킨 후
ㄴ .\venv\Scripts\Activate.ps1로 다시 활성화 시켜줌
ㄴ (venv) PS C:\Users\찬영\Desktop\Pyside GUI>  잘 나옴

가상환경 비활성화
ㄴ deactivate

가상환경 활성 상태 확인
ㄴ where python
ㄴ 가상환경이 활성화되면 일반적으로 터미널 프롬프트 앞에 (가상환경이름) 과 같은 표시가 붙습니다. 추가적으로 다음 명령으로 현재 파이썬 경로를 확인할 수 있습니다:

내가 만드는 프로젝트 조건

파이썬으로 gemini api를 사용하여 소설 릴레이를 만들려고 해 내가 조건을 적으면 너가 5줄 이내의 이야기를 만들어 주고 내가 그 이야기를 이어가고 그걸 너가 다시 이어가는 구조인거지 여기서 내가 이야기를 이어갈때마다 선택 기회를 사용할꺼야 그리고  서버와 클라이언트 통신을 이용할꺼야 너가 답변하는 쪽은 서버 코드로 짜고 내가 적는 부분들은 클라이언트 받아서 사용하면 될거 같아
GUI는 Pyqt6를 사용했어

조건 1. 처음에 장르를 적고 초반 설정을 적을꺼야 그럼 거기에 맞게 너가 5줄 정도 이야기를 풀어줘 그럼 내가 그 이야기를 이어갈께
조건 2. 선택 기회를 처음에 정해서 그 기회를 다 쓰면 엔딩을 보는 구조로 만들꺼야
조건 3. 초반 설정에서 애 먹을 수 있으니까 초반 설정 랜덤 버튼을 누르면 너가 알아서 5줄 정도 이야기를 만들어줘 물론 그 뒤는 내가 이어가고 내가 이어갈때 마다 선택 기회에서 1씩 감소시킬꺼야  
조건 4. 내가 위에 올린 UI를 참고해서 코드를 짜줘
조건 5. 엔딩도 너가 적어줘 일관성있고 조금 자연스럽게 말이야
조건 6. UI의 경우 첫 화면이 가장 왼쪽이 뜨고 확인 버튼을 누르면 첫 화면에 적은 내용을 ai가 인식해서 전달해주는거야 그럼 메인 화면이 나오는데 거기서 소설 릴레이 게임을 하는거지 그리고 기회를 다 쓰면 맨 오른쪽인 엔딩 화면으로 넘어가
